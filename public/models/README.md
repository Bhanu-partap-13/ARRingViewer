# 3D Models Directory

Place your 3D model files (.glb or .gltf format) in this directory.

## Required Models:

1. `ring-halo-twist.glb` - Halo Twist Diamond Ring
2. `ring-emerald-solitaire.glb` - Emerald Cut Solitaire
3. `ring-vintage-rose.glb` - Vintage Rose Gold Ring
4. `ring-three-stone.glb` - Three Stone Diamond Ring
5. `ring-cushion-halo.glb` - Cushion Halo Engagement Ring
6. `ring-pear-shape.glb` - Pear Shape Diamond Ring

## Format Requirements:

- File format: .glb (recommended) or .gltf
- Max file size: 10MB per model (for optimal loading)
- Texture maps: Include in the .glb file
- Scale: Real-world dimensions preferred

## AR Compatibility:

Models should be optimized for AR viewing:
- Clean geometry
- Optimized polygon count
- Proper scaling
- Correct orientation (ring facing upward)

## Creating/Converting Models:

You can use tools like:
- Blender (free) - Export as .glb
- Sketchfab - Download models in .glb format
- Online converters for other 3D formats
